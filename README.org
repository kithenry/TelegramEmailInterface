* Telegram To Mail

* App description
- A script to support sending email messages from the Telegram app.


* Extern Dependencies
- The telegram library

* Configuration
- Get an app password for your email.
- Create a bot with bot father and get its token.
- Populate the necessary fields in the config with the data from the previous fields.
- Run the script t2mail.py
- In telegram, send a message to the bot with the command /send_mail. As in
  - /send_mail message
    - where message follows the format : R: recepient_email| S: email_subject | C: email_content

* How it works
- User sends message in the format 
  - R: recepient@mail.com S: subject C: Content body
- Script breaks it down into
  R: recepient's email
  S: subject for email
  C: Email body
- Script creates an email object and sends it using the senders configured email via smtp.
